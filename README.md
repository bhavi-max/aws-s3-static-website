# AWS S3 Static Website Hosting

## Project Overview
This project demonstrates how to host a static website using Amazon S3. The website is publicly accessible and configured using proper bucket policies and public access settings.

## AWS Services Used
=> Amazon S3
=>  AWS IAM

## Project Features
=>  Static website hosting using S3
=>  Public access configuration
=>  Bucket policy for secure access
=> Cost-effective hosting solution

## Steps to Implement
1. Created an S3 bucket with a globally unique name
2. Enabled static website hosting in bucket properties
3. Uploaded static website files (HTML, CSS)
4. Disabled block public access for the bucket
5. Added a bucket policy to allow public read access
6. Accessed the website using the S3 website endpoint
